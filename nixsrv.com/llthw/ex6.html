Exercise 6. Bash: language settings, LANG, locale, dpkg-reconfigure locales

In Linux language selection is as simple as an exporting of a variable. That is right, programs decide how to talk to you by looking into this varibale. Of course, for this to work program must support locales and have translation to your language available and installed. Let us see how it works by installing French locale.

Now you will learn how to install and select a locale.

Do this

1: echo $LANG
2: locale
3: man man # press q to exit man
4: sudo dpkg-reconfigure locales

Now, select the fr_FR.UTF-8 locale by navigating through the list using cursor keys and select the locale using space. Select en_US.UTF-8 as the default system locale. 

 5: export LANG=fr_FR.UTF-8
 6: echo $LANG
 7: locale # press q to exit man
 8: man man
 9: export LANG=en_US.UTF-8

Explanation

  1. Prints out your current LANG variable which is used by programs to determine which language to use when interacting with you.
  2. Prints out all locale variables which are used by programs for setting up number, address, phone format, etc. according to the format of the specified country.
  3. Shows you manual page about unix manual system. Notice how I used # to comment an action, everything after # is not executed.
  4. Executes a program for re-configuring your locales. Because this change is system wide, you need to run this command as root, that's why there is sudo in front of dpkg-reconfigure locales. Don't bother about sudo now, I'd set it up for you.
  5. Exports LANG variable which is used to set all other locale variables.
  6. Prints out LANG variable, you can see that it's changed as you would expect.
  7. Prints out other locale variables which are changed also.
  8. Shows you man manual page, in French.
  9. Reverts LANG variable back to English.

Extra credit

  1. Read manual page about locale. To do this, type in man locale.
  2. Now, read a man 7 locale page. Notice how I used 7 here to call a man page about conventions. If you want, read man man now to understand what other possible codes are, or just wait for exercise which covers it.
